# 🌐 Students API

A RESTful Web API built with **C#** and **ASP.NET Core** for managing student data. This project was created as a learning exercise to explore ASP.NET Core development and implement common web security practices.

## 💻 Features

* CRUD operations for students
* RESTful API design
* HTTPS enforcement
* CORS configuration
* Authentication and Authorization
* Ownership-based & Policy-based Authorization
* Access & Refresh token
* Rate Limiting
* Input validation
* Logging & Auditing
* Clean and maintainable architecture

## 🔗 Technologies

* C#
* ASP.NET Core
* SQL Server
* JWT Authentication

## 🚀 Purpose

The main goal of this project is to gain hands-on experience with ASP.NET Core while learning how to secure modern web APIs using industry-standard security mechanisms and best practices.
